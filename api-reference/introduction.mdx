## Authentication

All API endpoints are authenticated using the VeyraX API Key. Include your API key in the header of all requests:

```bash
VEYRAX_API_KEY: YOUR_API_KEY
```

You can find your API key in your [account settings](https://www.veyrax.com/agent/new).
