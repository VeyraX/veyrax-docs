## Step 1: Get Your API Key

First, obtain your Veyrax API key from the dashboard.

## Step 2: Discover Available Tools

To see which tools are available to you:

```javascript
// Get all available tools
async function getAvailableTools() {
  const response = await fetch('https://api.veyrax.com/get-tools', {
    headers: {
      'VEYRAX_API_KEY': 'your_api_key_here'
    }
  });
  
  const data = await response.json();
  return data.tools;
}

// Usage
const tools = await getAvailableTools();
console.log(tools);
// Output: { "gmail": {...}, "slack": {...}, ... }
```

## Step 3: Call a Tool

To call any tool method:

```javascript
// Call a tool method
async function callTool(tool, method, params = {}) {
  const response = await fetch(`https://api.veyrax.com/tool-call/${tool}/${method}`, {
    method: 'POST',
    headers: {
      'VEYRAX_API_KEY': 'your_api_key_here',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify(params)
  });
  
  return await response.json();
}

// Example: Get Gmail messages
const gmailResult = await callTool('gmail', 'get_messages', {
  filters: {
    subject_keywords: 'important'
  }
});

// Example: Send a Slack message
const slackResult = await callTool('slack', 'send_message', {
  channel: "general",
  text: "Hello from Veyrax API!"
});
```

That's it! With just these few API calls, you can access over 20 powerful tools through a simple, consistent interface.
